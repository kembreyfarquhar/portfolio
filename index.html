Trip Split API

This project was built in 4 days on a remote team of 6 developers. I was solely in charge of creating the backend API. The product vision was an app that allows users to split expenses while on "trips" together. An easy way to have one person pay for an expense, alleviating the stressful task of trying to split bills in the moment, and keep track of the amount each person owes for that bill. This project was built using Node.js, Express.js, Knex.js, and utilizing a PostgreSQL database, hosted on Heroku.
